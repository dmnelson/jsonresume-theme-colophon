# Changelog

## 0.1.0 - 2026-06-05

- Initial public release of `jsonresume-theme-colophon`.
- Added a quiet, text-first JSON Resume renderer with inlined screen and print CSS.
- Supports the standard JSON Resume sections, defensive escaping, safe URL handling,
  and print/PDF styling.
- Publishes a serverless-compatible `dist/index.js` entrypoint with CSS embedded at
  build time.
